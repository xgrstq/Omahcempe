Website resmi Omahcempe — Solusi ternak & aqiqah syari berbasis di Kediri, Indonesia.
Versi ini merupakan pengembangan ulang dari website sebelumnya dengan pendekatan struktur yang lebih rapi, modular, dan fokus pada performa serta kemudahan penggunaan di perangkat mobile.
✨ Tujuan Project
Meningkatkan kredibilitas brand secara digital
Memberikan informasi layanan yang jelas & transparan
Mempermudah pelanggan dalam melihat pricelist dan melakukan pemesanan
Optimal untuk tampilan mobile
🚀 Fitur Utama
Landing page informatif
Section layanan & kategori ternak
Pricelist terstruktur
CTA langsung ke WhatsApp
Bottom navigation (mobile-first approach)
Deployment production via Vercel
🛠 Tech Stack
JavaScript
TailwindCSS
HTML
Vercel (deployment)
📦 Status
Production build — aktif digunakan dan terus diperbarui sesuai kebutuhan bisnis.
👨‍💻 Development
Dikembangkan sebagai project web komersial dengan pendekatan modular UI dan fokus pada performa serta maintainability.
